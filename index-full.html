    What is FYFYSWAP?

    FYFYSWAP is a non-custodial cross chain AMM-DEX protocol for swapping tokens based on the Fyfy (and Kusama) network(s), Parachains, and blockchains connected via a Bridge. The FYFYSWAP protocol effectively removes trusted intermediaries and provides the opportunity for faster trading. FYFYSWAP is open-source software licensed under Apache V2.

    —

    How does FYFYSWAP work?

    FYFYSWAP is a DEX with the possibility to set up multiple liquidity sources under a liquidity aggregation algorithm. When traders call the swap function, the liquidity aggregation algorithm will fill the order using the best offers from liquidity sources. Liquidity sources can be represented as AMMs, order books, or other algorithms. Because FYFYSWAP is an open-source project, it will be possible to add more liquidity sources as a community by developing inside the FYFYSWAP module.

    —

    Does the FYFYSWAP DEX have a token?

    Yes, FYFYSWAP does have a token called FYFYSWAP. FYFYSWAP enables users to create liquidity pools for token trading pairs in the FYFYSWAP DEX and receive FYFYSWAP: so-called liquidity staking. Trading fees are used to buy back and burn FYFYSWAP tokens, and new FYFYSWAP tokens are minted to reward liquidity providers, but this is done in a deflationary way as more FYFYSWAP are burned than minted. The FYFYSWAP token issuer will be a smart contract that will distribute FYFYSWAP according to programmed logic and to incentivize FYFYSWAP growth. Additionally, the FNT token will be used for paying network transaction fees. For a more in-depth description, check out the FYFYSWAP intro article.

In the course of development, the team will publish more information about the distribution of Fyfy swap tokens and we will update the community as soon as there is more news.

    What is the utility of the FYFYSWAP token?

    The FYFYSWAP token incentivizes liquidity providers, utilizing a deflationary model with respect to token supply. The 0.3% fee for every trade in the FYFYSWAP DEX is used to buy back FYFYSWAP tokens. Part of these FYFYSWAP tokens are allocated to the liquidity providers, and part are burned to decrease the token supply. At first, 100% of bought-back FYFYSWAP tokens are allocated to the liquidity providers, but with time this percentage decreases with the burned tokens.

    —

    How is Fyfy Network (FNT) used in the FYFYSWAP protocol?

    FYFYSWAP will be built on top of the FYFY Network, and the FYFY network token (FNT) will be used for gas/fees and liquidity provision on FYFYSWAP. FNT is a useful token for liquidity provision on FYFYSWAP because it is stable-ish (minimizing impermanent loss), highly liquid (because of the token bonding curve market maker), and neutral between other Fyfy ecosystem projects, but uses SOL and ETH as reserves.

    —

    How are prices determined on FYFYSWAP?

    The FYFYSWAP protocol design allows multiple liquidity sources per token pair. Every liquidity source will have its own algorithm to determine prices. When a user calls a swap function, the FYFYSWAP liquidity aggregation algorithm will fill the order using the best offers from liquidity sources.

    —

    Is FYFYSWAP fully permissionless?

    Yes! The FYFYSWAP protocol and traders cannot be censored or allowlisted. The developers cannot technically stop or edit the smart contracts in any way after they have been deployed. Deployment of developed code will be decided by the community and is not the decision of the developers.

    —

    What is a FYFYSWAP pool?

    There will be multiple, different pools in FYFYSWAP. It will start with a classic 50/50 XYK pool, and then add multiple other pool technologies to provide a better trading experience for users. FYFYSWAP will have a liquidity aggregation path algorithm, which will fill trades according to the best offers using multiple pools simultaneously.

    How can I use FYFYSWAP as a trader?

    There are four ways a trader can interact with with the FYFYSWAP protocol:
    —Through the FYFYSWAP DEX front-end (after launch)
    —Through the Fearless Wallet or the FYFY app (after launch)
    —Directly through smart contracts (after launch)

    How can I use FYFYSWAP as a liquidity provider?

    Liquidity providers have two ways to interact with the FYFY protocol:
    —Through the multiple liquidity pools manager front-end (after launch)
    —Directly through smart contracts (after launch)

    Are there risks in using FYFYSWAP?

    The FYFYSWAP protocol smart contract will be audited, and it is designed with security as its top priority. The code will be audited by a professional firm; however, we cannot guarantee that bugs will not be found in the future. Please do your own research and be careful.
